# MAS552-CS-RADAR-CURSOR
This project is a web-based radar chart generator designed for urban analytics, planning evaluation, and visual comparison of complex site performance indicators. It is inspired by the “Urban Performance” radar graphic and aims to transform that kind of static, highly designed visualization into a flexible digital tool that can be generated dynamically from structured backend data. The main idea is to create a front end that is not just a chart, but a reusable visual system capable of rendering presentation-quality radar diagrams for different projects, datasets, and research contexts. The tool should allow the user to feed a backend or JSON-based data model into the system and automatically generate a customized radar chart from that information.

A core objective of the project is full configurability. The user should be able to change the number of axes, the text of each axis, the KPI symbols, the mathematical formulas, the compared datasets, the colors, fills, line weights, legend labels, section titles, and footnotes. The radar should support multiple overlapping scenarios, such as “Existing Site,” “Proposed Site,” and “Barcelona,” while also handling a variable number of indicators without breaking the composition. In addition, the axis labels should support both plain text and LaTeX-rendered mathematical notation, so that each KPI can include a short symbolic label and a more formal equation when needed.

Visually, the project should feel closer to a scientific urban dashboard than to a generic charting interface. The aesthetic should be clean, dark, precise, and highly legible, with careful control over radial label placement, spacing, and hierarchy. The interface should include editing controls, live preview, and export options such as PNG, SVG, and JSON. Overall, this radar generator is intended to become a flexible front-end layer for urban performance analysis, making it possible to compare indicators such as proximity, diversity, density, and other custom metrics across sites, scenarios, or planning alternatives in a way that is rigorous, readable, and visually compelling.
